
#### 1. 项目简介

&nbsp;&nbsp;&nbsp;&nbsp;“PharosAdmin” 灯塔导航系统后台管理系统是一个基于Vue + TDesign框架开发的
公司内部导航系统后台，方便管理导航信息、导航类目信息等。</br>

```bash
## 安装依赖
npm install

## 启动项目
npm run dev
```

#### 4. 构建

```bash
## 构建正式环境
npm run build
```
