TDesign Vue Starter 是一个基于 tdesign-vue，使用 `Vue2`、`Webpack`开发，可进行个性化主题配置，旨在提供项目开箱即用的、配置式的初始项目。

<p>
  <a href="https://tdesign.tencent.com/starter/">使用文档</a>

</p>

### 开发

```bash
## 安装依赖
npm install

## 启动项目
npm run dev
```

### 构建

```bash
## 构建正式环境
npm run build
